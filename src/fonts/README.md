# Font Files Required

Please add your custom font files to this directory:

1. **Baron.otf** - The bold geometric font for hero titles and navigation
2. **Carousel.otf** - The elegant serif font for mission statements and quotes

## Font Usage

- **Baron**: Hero titles, navigation, section headers, CTAs
- **Carousel**: Mission statements, quotes, section subtitles
- **Inter** (system default): Body text, paragraphs, UI elements

## Note
If your font files have different extensions (.ttf instead of .otf), update the paths in `src/index.css` accordingly.
